# 🌀 Blind Maze

A multiplayer blind maze board game where players navigate a fog-of-war maze while a host watches everything unfold.

## How to Play

1. **Host** opens `/host.html` and creates a lobby
2. **Players** open the main page, enter their name, and join
3. Players navigate the maze using arrow keys or on-screen buttons — but can only see tiles they've already visited!
4. Land on **good tiles** to earn coins and expand the map, **bad tiles** for surprises, or find the **Exit** to win

## Features

- 2–6 players
- Fog-of-war mechanic (each player has their own map)
- Shop system with power-up items
- Dice-roll tile effects (D20)
- 8-bit procedural sound effects
- Live map editor for the host
- Customizable tile & player sprites
